### 区域参与

各地区的参与情况在美国略高，在欧洲略低于 2018 年，但与 2017 年的参与情况非常接近。

+   美国/加拿大，40.6%（对比去年 36.6%）

+   欧洲，33.3（对比 37.5%）

+   亚洲，12.1（对比 11.7%）

+   拉丁美洲，7.3（对比 6.6%）

+   非洲/中东，4.5%（持平）

+   澳大利亚/新西兰，2.3（对比 3.1%）
