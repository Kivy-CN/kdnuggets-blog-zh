## 为什么从 API 中收集数据？

你可能会想，为什么我们不直接使用 CSV 或从数据库中提取数据？有两个原因说明你应该学习 API：

**1.** API 是一种非常常见的行业专业数据收集方式，因此如果你将来成为数据科学家，你需要学习如何使用它。

**2.** 这比从数据库中提取数据的方式更复杂、更高级。因此，这也是学习 API 和给同事及招聘经理留下深刻印象的另一个理由。
