这是类似的 2016 年调查结果：![Top 10 algorithms/methods uses 数据科学家使用的顶级算法和方法](img/poll-algorithms-used-data-scientists.html)

另见一项非常有趣的 Kaggle 调查 [2017 年数据科学与机器学习现状](https://www.kaggle.com/surveys/2017)，其中还包括了一个关于算法的部分。

Kaggle 调查问到：工作中使用了哪些数据科学方法？顶尖的回答是

+   逻辑回归（Logistic Regression）

+   决策树（Decision Trees）

+   随机森林（Random Forests）

+   神经网络（Neural Networks）

+   贝叶斯技术（Bayesian Techniques）

+   集成方法（Ensemble Methods）

+   支持向量机（SVMs）

+   梯度提升机（Gradient Boosted Machines）

+   卷积神经网络（CNNs）
