# 使用这个预配置的 Python VM 镜像在几分钟内完成深度学习

> 原文：[https://www.kdnuggets.com/2017/05/deep-learning-pre-configured-python-vm-image.html](https://www.kdnuggets.com/2017/05/deep-learning-pre-configured-python-vm-image.html)

[Adam Geitgey](https://medium.com/@ageitgey) 在他的博客中写关于机器学习、深度学习、图像和语音识别及相关主题的文章。他涵盖了理论与实践，专注于开发者如何快速开始使用这些技术。Geitgey 说：

> 但我收到的头号问题是“**我到底怎么把这些开源库安装并运行在我的计算机上？**”

![Python 深度学习](../Images/6c538e457b1679a325463b2a107f510d.png)

为此，Geitgey 制作了一个虚拟机镜像，他在[这篇近期文章](https://medium.com/@ageitgey/try-deep-learning-in-python-now-with-a-fully-pre-configured-vm-1d97d4c3e9b)中进行了概述。该镜像基于 Ubuntu Linux Desktop 16.04 LTS 64 位，包含以下工具和库等：

+   Python 3.5

+   OpenCV 3.2 带有 Python 3 绑定

+   dlib 19.4 带有 Python 3 绑定

+   TensorFlow 1.0 for Python 3

+   Keras 2.0 for Python 3

+   Theano

+   face_recognition for Python 3（用于玩脸部识别）

它还包括一些预制项目，可以立即启动：

> 如果你从左侧边栏启动 PyCharm Community Edition，会有几个预先创建的项目可以打开。尝试 face_recognition、OpenCV 或 Keras 项目，并运行一些演示。右键点击代码窗口，选择“Run”以在 PyCharm 中运行当前文件。

阅读有关镜像的更多信息[这里](https://medium.com/@ageitgey/try-deep-learning-in-python-now-with-a-fully-pre-configured-vm-1d97d4c3e9b) —— 包括下载和安装说明、用户名和密码信息以及一些使用技巧 —— 并从[这里直接获取镜像](https://archive.org/details/DeepLearningVM_Ubuntu_16.04_64Bit_vmware)。
