“数据科学”这个术语最近才出现，用来专门指代一种新兴的职业，该职业预计将解释庞大的大数据存储。但理解数据有着悠久的历史，多年来科学家、统计学家、图书馆员、计算机科学家等都对此进行了讨论。

如今，数据科学作为一个业务领域非常复杂，由于其显著的受欢迎程度，对数据科学的描述众多，例如：

> 数据科学关注于分析数据并从中提取有用的知识。建立预测模型通常是数据科学家最重要的活动（Gregory Piatetsky, KDnuggets, https://www.kdnuggets.com/tag/data-science）。
> 
> 数据科学关注于分析大数据，以提取与可能性和误差估计的相关性（Brodie, 2015a）。
> 
> 数据科学是一门新兴学科，借鉴了统计方法学和计算机科学的知识，以便为广泛的传统学术领域创造有影响力的预测和洞察（哈佛数据科学倡议 https://datascience.harvard.edu）。

然而，用简单的话来说，**数据科学家只是试图从大量数据中获取洞察，帮助公司做出更明智的商业决策。**我们也将数据科学定义为一种从数据中推断出可操作洞察的方法。

数据科学使用广泛的数据导向技术，包括 **SQL**、**Python**、**R** 和 **Hadoop** 等。然而，它也广泛使用统计分析、数据可视化、分布式架构等技术，从数据集中提取意义。通过数据科学应用提取的信息用于指导业务流程和实现组织目标。

为了完成这一部分，我们还将提供数据挖掘、人工智能、机器学习和深度学习的简单定义，因为这些概念与数据科学及相互之间都有关系。

+   数据挖掘旨在理解和发现数据中新的、之前未曾见过的知识。

+   人工智能（AI）关注于使机器变得智能，旨在创建一个像人类一样行为的系统。

+   机器学习是人工智能的一个子集。机器学习旨在开发可以从历史数据中学习并通过经验改进系统的算法。

+   深度学习是机器学习的一个子集，其中数据通过多次非线性变换来计算输出。

### 数据科学简史

数据科学彻底改变了我们世界的多个方面。让我们来看看数据科学的起源和发展。

+   1962年，John W. Tukey 在《数据分析的未来》中写道——数据科学历史上的第一个里程碑被全球认可为杰出的美国数学家 John Tukey。John Tukey 在统计学领域的影响巨大，但他最著名的术语与计算机科学有关。事实上，他是第一个引入“bit”这一术语的，即“binary digit”的缩写。

+   1974年，Peter Naur 发表了《计算方法简明调查》，该调查涵盖了各种应用的数据处理方法。术语“数据科学”变得更加明确，因为他对其给出了自己的定义：“处理数据的科学，一旦数据被建立，与数据所代表的内容的关系则委托给其他领域和科学。”

+   1977年，国际统计计算协会（IASC）成立。

+   1989年，Gregory Piatetsky-Shapiro 组织并主持了第一次知识发现数据库（KDD）研讨会。

+   1994年，《商业周刊》发布了一篇关于“数据库营销”的封面故事。

+   1996年，在国际分类学会（IFCS）会议上，“数据科学”这一术语首次被纳入会议标题（“数据科学、分类及相关方法”）。同年，Usama Fayyad、Gregory Piatetsky-Shapiro 和 Padhraic Smyth 发表了《从数据挖掘到知识发现数据库》。

+   1997年，在他作为密歇根大学H.C. Carver统计学讲席教授的开幕讲座上，杰夫·吴呼吁将统计学改名为“数据科学”，将统计学家改名为“数据科学家”。

自21世纪初以来，数据存储量呈指数级增长，这主要得益于高效且具有成本效益的大规模处理和存储技术的出现。能够在“实时”中收集、处理、分析和展示数据和信息，为我们提供了前所未有的机会进行新的知识发现。为了处理这些海量数据，数据科学家需要高性能和大量技术组合，以在几秒钟内加速任务和数据处理。

像人工智能、机器学习和深度学习这样的颠覆性技术如今可以为数据科学家提供强大的平台。

### 实践数据科学的挑战

尽管分析的采用有所增加，但也带来了自身的一系列挑战。2017年，Kaggle对16000名数据专业人员进行的一项研究展示了他们在职业中面临的10大难题：

1.  脏数据 (36% 报告)

1.  缺乏数据科学人才 (30%)

1.  公司政治 (27%)

1.  缺乏明确的问题 (22%)

1.  数据不可访问 (22%)

1.  决策者未使用结果 (18%)

1.  向他人解释数据科学 (16%)

1.  隐私问题 (14%)

1.  缺乏领域专业知识 (14%)

1.  组织规模小，无法负担数据科学团队 (13%)

这些挑战显得尤为严峻。然而，我们需要认识到，每一次在新学科中的进步都会带来新的挑战。我们必须拥抱变革，确保这些变化有助于我们不断改进，获得新技能，扩展知识，探索新方法。

### 数据科学家是谁？

如前所述，随着操作数据的不断增长和新技术的不断涌现，我们越来越需要具有分析能力的专业人才，从大量数据中提取有价值的信息和见解，并做出准确的决策。我们称这种类型的专家为“数据科学团队”或简称“数据科学家”。

数据科学家是一位分析数据的专家，他们应该精通解决现代世界复杂问题所需的技术技能。如今的前沿技术，如人工智能、物联网、5G、机器人技术、区块链等，都高度依赖数据，只有能够操作数据并将其转化为有利可图的产品的人，才能引领未来的数字业务。

因此，数据科学家在每个公司和组织的业务发展战略中扮演着至关重要的角色。正如托马斯·H·德文波特和D.J.帕提尔所说，数据科学家是21世纪最性感的职业。

### 数据科学工具为数据科学家服务

提供了大量软件工具来支持数据科学家深入数据科学的世界。如今，现有的平台使数据科学家能够使用他们最熟悉的工具（如 Python、Jupyter 和 Dask）进行大规模工作。通常，这些服务通过安全且可扩展的基础设施提供，用于在云环境中运行数据科学和机器学习工作负载。数据团队可以在 Python 中大规模开发和部署数据科学模型，同时自动化 DevOps 和 ML 基础设施工程。

这些平台支持许多有用的 Python 库。

*“Python 库是一组函数和方法的集合，使数据科学家能够执行许多操作而无需编写代码。”*

+   **NumPy**是一个为 Python 编程语言提供支持的大型多维数组和矩阵的库。

+   **Seaborn**是一个基于 matplotlib 的 Python 数据可视化库。

+   **TensorFlow**是一个免费的开源软件库，用于在各种任务中进行数据流和可微分编程。

+   **PyTorch**是一个基于 Torch 库的开源机器学习库。

+   **Numba**是一个开源 JIT 编译器，它使用 LLVM 将 Python 和 NumPy 的子集翻译成快速的机器代码。

+   **SciPy**是一个免费的开源 Python 库，用于科学计算和技术计算。

+   **Pandas**是一个为 Python 编程语言编写的数据处理和分析软件库。

+   **Scikit-learn**是一个免费的 Python 编程语言的机器学习库。

+   **Matplotlib**是一个为 Python 编程语言及其数值数学扩展 NumPy 提供绘图功能的库。

+   **Bokeh**是一个在 Python 中提供高性能互动图表和绘图的数据可视化库。

### 我们要去哪里？观点。

正如约翰·图基所预测：“*数据分析的未来可能会有巨大的进步，克服实际困难，并为所有科学和技术领域提供巨大的服务*。” 在过去的几年里，我们见证了许多数据驱动的技术创新，例如5G闪电般的互联网速度、机器学习、云计算以及区块链概念，这些只是一个值得注意的名单，远非详尽。数据的爆炸以及不断增长的技术能力只是开始，我们的生活正在变得“更智能”，随着技术创新很快可能会融入人类生活的各个方面。
