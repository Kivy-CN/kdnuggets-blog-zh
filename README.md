## 下载

### Docker

```
docker pull apachecn0/kdnuggets-blog-zh
docker run -tid -p <port>:80 apachecn0/kdnuggets-blog-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g kdnuggets-blog-zh
kdnuggets-blog-zh <port>
# 访问 http://localhost:{port} 查看文档
```
